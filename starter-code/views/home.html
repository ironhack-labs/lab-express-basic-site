What it is?

Express is a little framework that help us to create a server inside our machine and save page and resources like this page.

Steps!

To uses Express in our project we have to follow some specific steps!

  • Install Express inside our project with npm

    npm install express --save

  • Get the resources

    var app = request(express)

  • Set the requests

    agg.Get('/',newPage.html)

  • And finally listen the requests

    app.listen('port')
